    		Automated Landslide Event Archive Generation from News Articles
    Rishideep Chatterjee, Subhayan Bhattacharya, Dr. Sarbani Roy.
    IEEE ASIANCON 2023  •  Acknowledgement

    Cite as - S. Bhattacharya, R. Chatterjee and S. Roy, "Automated Landslide Event Archive Generation from News Articles," 2023 3rd Asian Conference on Innovation in Technology (ASIANCON), Ravet IN, India, 2023, pp. 1-8, doi: 10.1109/ASIANCON58793.2023.10270576.


  • The primary challenge of building an archive is the variety of information sources and format. There are challenges in processing information from each of these sources.
  • As a part of this work, a novel framework for generating landslide archive is proposed, source of data archive are online news articles from dependable news publishers.
  • BERT QA model used is configured with questions about the date of the landslide, location of the landslide, and the number of people affected in the landslide.
  • For predicting the severity of the landslide event, 800+ news articles were annotated manually into classes High, Low, and Uncertain. Six classifiers (namely, Multi Layer Perceptron Classifier, Decision Tree Classifier, Random Forest Classifier, Support Vector Classifier, K Nearest Neighbour Classifier, and Stochastic Gradient Descent Classifier) were considered.
  • Stochastic Gradient Descent (SGD) classifier has the most promising results across all metrics, except Precision (Support Vector Machine has a marginally higher value) and Execution Time (Decision Tree has a lower value).
  • The end-to-end framework presented in this paper has a user-friendly web interface that allows running queries based on time and specific location of the landslide, and provides the information in a visualisable format.
  • The framework can be easily extended for other events and keywords as well.

    		Noise-resistant Local Binary Pattern (LBP) Descriptor

    Repository (research paper under preparation)

  • A powerful tool for texture recognition, Local Binary Pattern (LBP) descriptor is often sensitive to image noise, compromising visual quality & recognition accuracy.
  • The project presents a noise-resistant LBP descriptor for robust texture recognition through image processing - the approach involves four stages:
    (1) image preprocessing,
    (2) LBP feature extraction,
    (3) noise-resistant thresholding, and
    (4) texture pattern classification using a Machine Learning algorithm
  • Experimental results on the Brodatz texture dataset demonstrate the superiority of the method in achieving enhanced recognition accuracy compared to traditional LBP approaches.
  • Algorithm used: Support Vector Machine (SVM)
  • Tech Stack: Python CV2, Scikit-learn, Numpy
